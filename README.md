# 书法字体制作器

> 将手写字体转换为可使用的 TTF 字体文件，由 **谷子** 研发。

## 功能介绍

- **单字上传** — 支持拖拽、点击选择或直接粘贴（Ctrl+V / Cmd+V）手写单字图片
- **智能预处理** — 自动裁剪、居中、二值化，将手写图片标准化为 1024×1024 字模
- **人工标注** — 上传后显示预处理效果，输入对应的标准汉字确认收录
- **实时预览** — 收录的字体自动生成预览，与系统楷体对照查看效果
- **一键下载** — 导出标准 TTF 字体文件，可直接安装到系统使用

## 使用流程

1. 创建一个项目
2. 逐个上传手写字图片（PNG / JPG）
3. 在待确认区域输入每个字对应的标准汉字，点击确认
4. 左右对照预览效果
5. 下载 TTF 字体文件

## 快速开始

```bash
# 安装依赖
pip install -r requirements.txt

# 启动服务
python run.py
```

打开浏览器访问 http://127.0.0.1:8000

## 技术栈

- **后端**: FastAPI + Pillow + scikit-image
- **前端**: 原生 HTML/CSS/JS
- **字体生成**: fonttools
